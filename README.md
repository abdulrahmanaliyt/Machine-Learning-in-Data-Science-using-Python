## 📘 About

This repository is a **complete practical companion** to the textbook *"Machine Learning in Data Science Using Python"* by Dr. R. Nageswara Rao. It converts every theoretical concept in the book into clean, executable Python code — making it ideal for students, self-learners, and professionals who want to build real intuition for ML and Data Science.

Each chapter is self-contained with working scripts, Jupyter notebooks, and practice datasets so you can learn by doing.

## 🚀 Getting Started

### Prerequisites

Make sure you have Python 3.8 or higher installed. You can check with:

```bash
python --version
```

### Installation

**1. Clone the repository**

```bash
git clone https://github.com/your-username/Machine-Learning-in-Data-Science-using-Python.git
cd Machine-Learning-in-Data-Science-using-Python
```

**2. Create a virtual environment (recommended)**

```bash
python -m venv venv
source venv/bin/activate        # On Windows: venv\Scripts\activate
```

**3. Install dependencies**

```bash
pip install numpy pandas matplotlib seaborn scikit-learn tensorflow keras jupyter
```

**4. Run a chapter script**

```bash
cd "Part 2 - Machine Learning in Data Science/Chapter 19 - Simple Linear Regression"
python simple_linear_regression.py
```

Or launch Jupyter for interactive notebooks:

```bash
jupyter notebook
```
