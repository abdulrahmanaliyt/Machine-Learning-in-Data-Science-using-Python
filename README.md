# 🛠️ Prerequisites & Installation


## Requirements

- Python 3.10+
- Jupyter Notebook
- Git


## Clone Repository

```bash
git clone https://github.com/abdulrahmanaliyt/Machine-Learning-in-Data-Science-using-Python.git
```


## Open Folder

```bash
cd Machine-Learning-in-Data-Science-using-Python
```


## Install Libraries

```bash
pip install numpy pandas matplotlib seaborn scikit-learn tensorflow keras opencv-python
```
